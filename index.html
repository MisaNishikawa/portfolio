予約カレンダー

ペアでGithubを使いながら、予約カレンダーのページを作成しました。 各日付の予約ボタンを押すと、予約ページへ飛び、(削除ボタンは機能しておりません。) 予約ページで必要な項目を入力すると、データベースに入力内容が保存され、カレンダーページへリダイレクトという仕組みです。 企画→設計→環境構築→実施 という流れで行いました。
[言語:PHP　DB:MySQL]
GitHub URL↓ https://github.com/Naction/original

To Doアプリ

一覧ページからcreateボタンを押す→タスクの新規追加画面へ移動→ フォームを入力し、POSTボタンを押す→ 入力したタスクが追加された一覧ページにリダイレクトされ、入力内容がデータベースに保存される→ EDITボタンを押すと編集画面に移動し、編集が可能。DELETEボタンを押すとタスクが削除されます。
[言語:PHP　DB:MySQL]
GitHub URL↓ https://github.com/MisaNishikawa/todoo2
